# 2. Техническое задание

## Цели разработки
- Создать мобильное приложение для управления питанием
- Автоматизировать процесс планирования рациона
- Обеспечить взаимодействие между тренерами и клиентами

## Описание системы
**FitRation** — мобильное приложение для контроля питания с элементами социальной сети. Пользователи могут отслеживать свой рацион, получать автоматически сгенерированные планы питания и привязываться к тренерам для профессионального руководства.

## Целевая аудитория
- Обычные пользователи — люди, следящие за питанием
- Спортсмены — нуждающиеся в точном расчёте КБЖУ
- Диетологи/тренеры — профессионалы, консультирующие клиентов

## Общие требования
- Кроссплатформенность (Android)
- Интуитивный интерфейс
- Защита персональных данных
