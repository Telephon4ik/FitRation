# 3. Требования

## Функциональные требования

### Для всех пользователей
- FR1: Регистрация и аутентификация
- FR2: Редактирование профиля
- FR3: Просмотр своего рациона
- FR4: Отметка съеденных блюд

### Для пользователей (USER)
- FR5: Генерация дневного рациона
- FR6: Отправка заявки тренеру
- FR7: Просмотр комментариев тренера

### Для тренеров (COACH)
- FR8: Просмотр списка клиентов
- FR9: Поиск пользователей по ID
- FR10: Отправка заявок пользователям
- FR11: Просмотр и обработка входящих заявок
- FR12: Просмотр рациона клиентов
- FR13: Добавление комментариев к блюдам

## Нефункциональные требования

### Производительность
- NFR1: Время загрузки экрана < 2 секунд
- NFR2: Генерация рациона < 5 секунд
- NFR3: Отклик UI < 100 мс

### Безопасность
- NFR4: Шифрование паролей
- NFR5: Валидация входных данных
- NFR6: Защита от SQL-инъекций (Firestore security rules)

### Удобство использования
- NFR7: Соответствие Material Design
- NFR8: Адаптация под разные размеры экранов
- NFR9: Интуитивная навигация

### Надёжность
- NFR10: Доступность 99.5%
- NFR11: Восстановление после сбоев
- NFR12: Обработка ошибок сети

### Совместимость
- NFR13: Android 8.0+ (API 26)
